# Анализ подлинности лиц и детекция фейков

## Описание работы

Данная научная статья посвящена анализу методов **детекции фейковых изображений лиц** с применением алгоритмов искусственного интеллекта и машинного обучения. В работе рассматриваются:

- особенности формирования фейковых изображений
- методы выявления подделок
- сравнительный анализ классических и современных подходов
- оценка устойчивости моделей к различным типам искажения

Статья представляет теоретическое обоснование проблематики и практические выводы по эффективности различных методов

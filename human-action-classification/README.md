# Исследование: Классификация человеческих действий

**Описание работы:**  
Эта научная статья посвящена анализу возможности классификации человеческих действий с использованием методов машинного обучения. В работе рассматриваются:

- постановка задачи распознавания действий человека
- обзор подходов и моделей
- методы извлечения признаков
- анализ эффективности разных алгоритмов

**Содержание папки:**

Компьютерное зрение, Детекция человеческих действий, Распознавание человеческих действий, 
CNN, ResNet50, Xception, DenseNet169

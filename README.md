# AI Computer Vision Research

Данный репозиторий содержит научные исследования в области  
**компьютерного зрения** и **искусственного интеллекта**.

Работы посвящены анализу и применению методов машинного обучения
для решения задач распознавания и верификации визуальных данных.

---

## Структура репозитория

Репозиторий организован по тематическим папкам, каждая из которых
содержит отдельную научную работу с подробным описанием и материалами

### 1. Human Action Classification
`human-action-classification/`

Исследование, посвящённое классификации человеческих действий
с использованием методов машинного обучения и компьютерного зрения

**Тематика:**
- анализ подходов к распознаванию действий человека  
- обработка визуальных данных  
- сравнительный обзор методов классификации  

Полное описание и материалы - в `README.md` внутри папки

---

### 2. Face Fake Detection
`face-fake-detection/`

Научная работа, посвящённая детекции фейковых изображений лиц
и анализу подлинности с применением методов искусственного интеллекта

**Тематика:**
- детекция поддельных изображений лиц  
- анализ устойчивости моделей  
- применение ИИ в задачах верификации  

Полное описание и материалы - в `README.md` внутри папки

---

Каждая папка содержит собственный `README.md` с подробным описанием
работы, структурой файлов и ссылками на публикации
